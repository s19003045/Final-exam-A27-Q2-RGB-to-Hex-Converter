# RGB - Hex Converter

RGB 與 HEX 色碼的轉換器

# 功能說明

- 〈RGB input〉〈RGB slider〉〈HEX input〉三者皆可使用，並連動變化
- 〈RGB input〉與 其 R 色塊、G 色塊、B 色塊連動變化
- 〈RGB slider〉與 SLIDER 之 R 色塊、G 色塊、B 色塊連動變化
- 〈HEX input〉與 HEX 色塊連動變化
